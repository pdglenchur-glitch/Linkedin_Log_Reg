# Linkedin User Prediction Tool 

This project predicts the probability that an individual is a Linkedin user depending on a series of demographic factors. This application was created as part of a project for the Georgetown MSBA program, utilizing public data from Pew (to be used for educational and training purposes only). 

------
## Usage

This application utilizes a logistic regression model trained on survey data from the Pew Research Center. Users can select from a series of demographic details within the application's sidebar to predictt whether or not the party in question is a Linkedin user. The user will also be able to see a breakdown of the model's performance, as well as relative visualizations broken down by each demographic variable.
